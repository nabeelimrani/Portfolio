Book Portal Project Portfolio Detail

A book portal project designed to provide users with a platform to discover, search, and manage their favorite books.

Project information

  • Category: Book Portal
  • Client: BXTrack Solutions
  • Project date: March, 2023
  • Project URL: https://github.com/nabeelimrani/bookportal.git

Key Features:

  • Book search and browsing functionality
  • User registration and profile management

  • Book wishlist and reading history

Technologies Used:

  • Database: MySQL
  • Framework: Laravel

Results and Impact:

Personal Contributions:
